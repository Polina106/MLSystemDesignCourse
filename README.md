## Personal diet assistant - Приложение для составление персональеого плана питания

Приложение, которое подбирает для вас план питания, построенный на основе ваших целей — будь то снижение веса, набор мышечной массы или просто поддержание формы. Просто заполните анкету, а умный алгоритм точно рассчитает вашу норму калорий и предложит персонализированное меню на каждый день.

### Задачи, решаемые в проекте

1. Разметка данных на основные виды аллергенов
2. Разработка рекомендательной системы, выдающей разнообразные блюда на основе предпочтений 
3. Разработка оптимизатора, действующего в условиях различного рода ограничениях (калории, аллергены, повторяемость блюд, время приготовления)

### Основные возможности приложения

1. Составление персонального плана рациона на основе заполненной анкеты предпочтений
2. Расчет калорийности блюд, исходя из выбранной стратегии 
3. Учет аллергенов и предпочтений пользователя (время готовки)
4. Составление плана питания на день или на неделю
5. Составление списка продуктов, необходимых для рецептов

### Технологический стек

Планируемый стек:
- *Python 3.10+*
- *FastAPI, Pydantic, Uvicorn* - Backend
- *PostgreSQL* - базы данных
- *CatBoost, surprise* - рекомендательная система
- *ruff* - линтер 
- *pre-commit* — запуск проверок перед коммитом

### Запуск
#### Требования
- Python 3.10+
- Git

#### Установка зависимостей
```bash
pip install ruff
```

#### Установка pre-commit хуков
```bash
pip install pre-commit
pre-commit install
```

#### Запуск проверки
```bash
ruff check .
ruff format .
```
